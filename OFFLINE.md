# Офлайн функціональність

Тренажер для скорочитання тепер повністю готовий до роботи офлайн на вашому комп'ютері.

## Що працює офлайн:

✅ **Повна функціональність для форматів:**
- `.txt` — текстові файли
- `.pdf` — PDF документи
- `.docx` — Microsoft Word документи  
- `.epub` — E-book файли

✅ **Всі функції приложення:**
- Завантаження й читання файлів
- Налаштування швидкості читання (WPM)
- Контроль анімації (пауза, розмір шрифту)
- Збереження стану читання
- Перемикання мови (українська/англійська)
- Система кешування (PWA)

## Що вимагає інтернету:

⚠️ **DJVU файли** — завантажуються з CDN (рідко використовується)

## Як використовувати офлайн:

### На комп'ютері (Windows/Mac/Linux):

1. **Завантажте файли на комп'ютер** (в папку з файлом `index.html`)
2. **Запустіть локальний сервер:**
   ```bash
   cd /path/to/speed-reading-trainer
   python3 -m http.server 8000
   ```
3. **Відкрийте у браузері:** `http://localhost:8000`
4. **Встановіть як PWA додаток** (браузер запропонує)
5. **Користуйтеся офлайн** без інтернету

### На мобільному (Android):

1. **Відкрийте програму в Chrome**
2. **Натисніть меню → Встановити додаток**
3. **Користуйтеся як звичайна программа**

## Структура файлів:

```
speed-reading-trainer/
├── index.html           # Основна сторінка
├── script.js           # Логіка приложення
├── styles.css          # Стилі
├── sw.js               # Service Worker для офлайн кешування
├── manifest.json       # Метаїнформація PWA
└── lib/                # Локальні бібліотеки
    ├── pdf.min.js
    ├── pdf.worker.min.js
    ├── mammoth.browser.min.js
    └── jszip.min.js
```

## Технічні деталі:

- **Service Worker** автоматично кешує всі файли при першому завантаженні
- **localStorage** зберігає налаштування й прогрес читання
- Програма працює навіть якщо з'єднання інтернету випаде під час використання
- Версія кешу: `speed-reading-trainer-v2`

## Оновлення офлайн кешу:

Якщо ви внесли зміни до файлів:
- Очистіть кеш браузера (або перезавантажте Service Worker)
- Або змініть номер версії в `sw.js` на більший
